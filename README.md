# Features

- Random clicks

- Do some game action key (A,W,S,D and space)

- Interactive GUI with random window title

# Config
**config.py** in assets folder

You should change ```"keys": ['a','s','d','w',' ']``` , ```"buttons": ['left']``` and ```self.SLEEP_TIME = 2```
